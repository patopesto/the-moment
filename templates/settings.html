🏷️ NFC Spool Locations

These Spoolman location names are shared by the NFC workflow and the settings below. When a spool is finished, tapping its tag and choosing Trash moves it to Trash so the tag can be re-programmed. Each location is created in Spoolman automatically the first time it is used.
When enabled, The Moment keeps Spoolman's spool location field in sync with toolhead assignments — in both directions. Assigning a spool writes {Printer Name} - T{index} (e.g. "Ender3 - T0") to Spoolman. Unassigning, deleting a printer, or reducing toolhead count moves the spool to the Inventory location. Spoolman is also polled every 5 minutes: editing a spool's location directly in Spoolman will update the toolhead assignment here automatically. Changes save immediately.
How long the first NFC tap stays pending while waiting for a second tap (5–120 s, default 15). Changes save immediately.
Where spools go when removed from a toolhead via NFC or location sync.
Where finished/empty spools go. Tag is ready for re-programming.
Location format written to Spoolman: {Printer Name} - T{toolhead index}  — e.g. "Roci - T0", "Ender3 - T0"

🔄 Spool Displacement

Covers one specific scenario: assigning a spool to a toolhead that already has one. When enabled, the displaced spool is automatically moved to the Inventory location. Works independently of bidirectional location sync.
