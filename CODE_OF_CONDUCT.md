# Code of Conduct

## Our Standard

The Moment is a maker-community project. Everyone who participates — in issues, pull requests, discussions, or any other forum — is expected to:

- Be respectful and constructive in all communication
- Focus feedback on ideas and code, not on people
- Welcome contributors of all skill levels
- Assume good intent; ask for clarification before assuming the worst

## Not Welcome

- Personal attacks or insults
- Deliberate intimidation or trolling
- Sustained disruption of discussions
- Publishing someone's private information without permission

## Scope

This code of conduct applies to all project spaces: GitHub issues, pull requests, discussions, and any other venue where this project is represented.

## Enforcement

Violations can be reported to the maintainer via the contact on the [GitHub profile](https://github.com/ThetaSigmaLabs). Reports are reviewed confidentially. Responses may include a warning, temporary ban, or permanent removal from the project — depending on severity and pattern.

## Credit

This project follows the spirit of the [Contributor Covenant](https://www.contributor-covenant.org), adapted for a small open-source project.
